sumarsono.id (domain sudah mati, hehe) ini dibangun menggunakan Jekyll, maka saya perlu setup jekyll untuk mesin Arch Linux saya. Berikut ini, langkah-langkahnya:

  1. Install ruby karena kita butuh RubyGems:

    sudo pacman -S ruby

  2. Selanjutnya edit .bashrc dan tambahkan:

    PATH="$(ruby -e 'print Gem.user_dir')/bin:$PATH"

  3. restart terminal

  4. Update paket gem:

    gem update

  5. Install bundler dan jekyll

    gem install jekyll bundler

  6. Masuk ke direktori static site milik kita dan install paket yang diperlukan:

    bundle install

  7. Setelah selesai, coba run static site kita secara lokal

    bundle exec jekyll serve

Kalau berhasil run, berarti sudah beres. :+1:
