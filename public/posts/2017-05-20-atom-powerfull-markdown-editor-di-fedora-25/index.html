Dulu, kali pertama mencoba Atom sewaktu awal-awal rilis. Rasanya tidak enak jika dibandingkan text editor lainnya. Setelah sekian lama, akhirnya hari ini saya unduh Atom dan mencoba lagi, beda sekali rasanya, sekarang lebih smoot.

Alasan kenapa hari ini mencoba Atom adalah untuk mencoba menggunakannya debagai Markdown editor. Ternyata lumayan ciamik. Fitur yang saya suka adalah markdown preview, jadi bisa melihat langsung hasil markdown yang kita tulis.

Untuk mengaktifkan fitur tersebut cukup klik Packages > Markdown Preview > Toggle Preview atau dengan shortcut Ctrl + Shift + M
