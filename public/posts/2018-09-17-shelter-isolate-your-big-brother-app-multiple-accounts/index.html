Hari ini secara tidak sengaja saya menemukan sebuah aplikasi yang menarik. Berawal dari ketika saya mengunjungi /r/LineageOS/. Disana ada yang merekomendasikan sebuah aplikasi, yakni Shelter. Apa itu Shelter? berikut ini saya kutip dari deskripsi di repo f-droid: > Shelter is a Free and Open-Source (FOSS) app that leverages the “Work Profile” feature of Android to provide an isolated space that you can install or clone apps into.

Shelter comes with absolutely no advertisement / statistics / tracking SDKs bundled with it. All source code is available in this public Git repository and the sources are licensed under WTFPL.

Menarik bukan? Oh iya, bagi pengguna fitur app clone khas Miui, pasti akan suka. Saya sudah mencoba Shelter di HP dengan sistem operasi dari LineageOS. Belum ada sehari sih, tapi so far so good.
